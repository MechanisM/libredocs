Liberate your ideas

Libre Docs runs only the application. We neither store your data nor track you, everything runs in your browser.s For this to work, we support the open remoteStorage protocol so you can have your data where you want.

For the applications, we use existing open web technologies like Etherpad, WebODF, SVG-edit and more.

Your own remote storage

On sign up you will get your own CouchDB at Iris Couch. This is only one of the options but we recommend Iris Couch because they are nice people who care about the open web and decentralization. You can always start with this option and switch to another later:

  • CouchDB: Install your own and run Couch Pimper against it.
  • ownCloud: Install it on your own server (pay special attention to point 4, as well as to the last paragraph).
  • Develop your own software or make existing software compatible with the remoteStorage protocol.
  • You can also connect your remote storage to WordPress and Friendica with the plugins.
